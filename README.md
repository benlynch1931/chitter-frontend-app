# Chitter



## How to install

1. Fork and Clone this repo.
2. Run in terminal:
```shell
npm install
```

## How to run

1. In terminal, run:
```shell
npm start
```
2. Head to `localhost:8080` in your browser


## Testing 

1. In terminal, run: 
```shell
open SpecRunner.html
```
2. This should automatically open Jasmine in your browser
